Ru: Задание 1
Код писать в файле script.js

1. Когда пользователь кликает на кнопку #b1, получить XML из файла /data/xml.xml и добавить содержимое элементе <shop_name> в контейнер #xmlContent, используя меотод $.ajax().

2. Когда пользователь кликает на кнопку #b2, получить HTML из файла /data/html.html и добавить его в контейнер #htmlContent с помощью метода .load(). 

3. Когда пользователь кликает на кнопку #b3, получить JSON из файла /data/json.json и добавить значение ключа а в контейнер #jsonContent, используя метод $.ajax().

4. Когда пользователь кликает на кнопку #b4, получить скрипт JavaScript из файла /data/js.js, используя $.ajax() и выполнить функцию myAlert().

5. Для всех запросов, используя $.ajaxSetup, установить глобально опции:
-type: "GET"
-global: true  

6. Для всех запросов реализовать глобальные обработчики событий: 
$(document).on({
	"ajaxStart": function() {
		// Показать #log с эффектом возникновения за 1c.
	},

	"ajaxComplete": function(event, xhr, ajaxOptions) {
		// добавлять в #log фразу "Request to " + тут надо получить ссылку запроса + " completed"
	},

	"ajaxStop": function() {
		// Спрятать #log c эффектом растворения за 1c
	}
});

7. Для запроса на получение XML отключить глобальные обработчики.
