Ru: Задание 2. (Домашнее задание) 
Код писать в файле script.js.

Реализовать обработчик события submit для формы, который должен выполнять следующие действия:
1. Запретить отправку формы стандартным способом.
2. Если предыдущий ajax запрос не завершился, то прервать его (метод .abort())
3. Собрать данные из формы с помощью метода .serialize()
4. Все поля формы (input, select, button, textarea) на время выполнения запроса сделать недоступными
5. Отправить ajax-запрос с помощью метода $.ajax, задав следующие параметры:
	- адрес запроса http://www.vitava.com.ua/files/jquery/ajax-scripts/send-data.php (Содержимое этого файла можно посмотреть в файле send-data.php)
	- тип данных от сервера jsonp
	- даныне для сервера: собранные данные формы
6. Реализовать обработик для колбека .done(), который должен выводить в консоль полученный с сервера ответ
7. Реализовать обработик для колбека .fail(), который должен выводить в консоль значения textStatus, errorThrown
8. Реализовать обработик для колбека .always(), который должен делать доступными для ввода поля формы
